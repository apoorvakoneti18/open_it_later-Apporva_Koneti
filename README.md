# Digital Time Capsule

**Digital Time Capsule** is a modern web application that allows users to securely preserve messages, photos, videos, and goals that unlock on a future date. Users can create private, public, or shared time capsules and track countdowns to relive memories intentionally.  

> *Preserve today. Unlock tomorrow.*

---

## Features

- **User Authentication** – Sign up and log in securely.  
- **Create Capsules** – Add title, description, and future unlock date.  
- **Upload Content** – Save text, images, and videos in your capsules.  
- **Privacy Settings** – Set capsules as private, public, or shared.  
- **Countdown Timer** – Track time remaining until a capsule unlocks.  
- **Explore Public Capsules** – Browse featured capsules shared by others.  
- **Responsive Design** – Works beautifully on desktop and mobile.  
- **Modern UI** – Dark mode, glassmorphism panels, subtle animations.  

---

## Tech Stack

- **Frontend:** React, TypeScript, CSS  
- **Backend / Storage:** Firebase or Supabase (for auth and file storage)  
- **Deployment:** Vercel / Netlify  
- **Design:** Glassmorphism, dark cosmic theme, animated backgrounds
